Struktura dokumentu HTML

⏱️ 20 minut 🟢 Łatwy Lekcja 2/12

Podstawowa anatomia dokumentu HTML

Każdy dokument HTML ma określoną strukturę, która jest niezbędna do prawidłowego działania strony. Poznajmy szczegółowo każdy element tej struktury.

🎯 Cele tej lekcji

  • Zrozumienie roli każdego elementu struktury HTML
  • Poznanie sekcji HEAD i jej zawartości
  • Zrozumienie sekcji BODY
  • Nauka o DOCTYPE i jego znaczeniu

DOCTYPE - Deklaracja typu dokumentu

DOCTYPE to pierwsza linia każdego dokumentu HTML5, która informuje przeglądarkę o wersji HTML.

Deklaracja DOCTYPE dla HTML5:

<!DOCTYPE html>

Dlaczego DOCTYPE jest ważny?

  • Informuje przeglądarkę, jak interpretować kod
  • Zapewnia zgodność z standardami web
  • Aktywuje tryb standardowy przeglądarki
  • Bez DOCTYPE przeglądarka może przejść w tryb quirks

Element <html>

Element <html> to kontener najwyższego poziomu, który otacza całą zawartość strony.

Element html z atrybutem lang:

<html lang="pl">
    <!-- Cała zawartość strony -->
</html>

Atrybut lang:

  • lang="pl" - określa język główny strony
  • Pomaga czytelnikom ekranu i wyszukiwarkom
  • Wspiera funkcje przeglądarki jak tłumaczenie
  • Poprawia dostępność strony

Sekcja <head> - Metadane dokumentu

Sekcja HEAD zawiera informacje o dokumencie, które nie są wyświetlane na stronie, ale są ważne dla przeglądarki.

Przykładowa sekcja HEAD:

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Tytuł strony</title>
    <meta name="description" content="Opis strony">
    <link rel="stylesheet" href="style.css">
</head>

Elementy sekcji HEAD:

<meta charset="UTF-8">

Określa kodowanie znaków - UTF-8 obsługuje wszystkie języki

<title>

Tytuł strony wyświetlany w zakładce przeglądarki

<meta name="viewport">

Kontroluje zachowanie strony na urządzeniach mobilnych

<link rel="stylesheet">

Łączy plik CSS ze stylem strony

Meta tag viewport

Meta tag viewport jest kluczowy dla responsywności strony na urządzeniach mobilnych.

Standardowy viewport dla responsywnych stron:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Parametry viewport:

  • width=device-width - szerokość równa szerokości urządzenia
  • initial-scale=1.0 - początkowy poziom powiększenia
  • user-scalable=no - blokada powiększania (opcjonalne)
  • maximum-scale=1.0 - maksymalny poziom powiększenia

Sekcja <body> - Zawartość strony

Sekcja BODY zawiera całą widoczną zawartość strony - tekst, obrazy, linki, formularze, itp.

Przykładowa sekcja BODY:

<body>
    <header>
        <h1>Główny nagłówek strony</h1>
        <nav>Menu nawigacyjne</nav>
    </header>

    <main>
        <h2>Treść główna</h2>
        <p>Paragraf z tekstem...</p>
    </main>

    <footer>
        <p>Stopka strony</p>
    </footer>
</body>
💡 Wskazówka

Używaj semantycznych tagów HTML5 jak header, main, nav, footer do lepszej struktury

🛠️ Ćwiczenie praktyczne

Zadanie 2: Stwórz kompletną strukturę HTML

Utwórz plik z pełną strukturą HTML zawierającą wszystkie omówione elementy:

<!DOCTYPE html>
<html lang="pl">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Moja druga strona HTML</title>
    <meta name="description" content="Strona demonstrująca strukturę HTML">
</head>
<body>
    <header>
        <h1>Witaj na mojej stronie!</h1>
        <nav>
            <a href="#o-mnie">O mnie</a>
            <a href="#kontakt">Kontakt</a>
        </nav>
    </header>

    <main>
        <section id="o-mnie">
            <h2>O mnie</h2>
            <p>Jestem [TWOJE IMIĘ] i uczę się HTML w TEB Edukacja.</p>
        </section>

        <section id="kontakt">
            <h2>Kontakt</h2>
            <p>Możesz skontaktować się ze mną przez email.</p>
        </section>
    </main>

    <footer>
        <p>&copy; 2025 Moja strona. Stworzona w ramach kursu HTML.</p>
    </footer>
</body>
</html>
Kroki do wykonania:
  1. Skopiuj kod do nowego pliku
  2. Zapisz jako struktura.html
  3. Zamień [TWOJE IMIĘ] na swoje imię
  4. Otwórz w przeglądarce
  5. Sprawdź tytuł w zakładce
  6. Przetestuj linki nawigacyjne

📝 Podsumowanie

Czego się nauczyłeś:

  • DOCTYPE html deklaruje użycie HTML5
  • Sekcja HEAD zawiera metadane niewidoczne dla użytkownika
  • Meta charset="UTF-8" zapewnia poprawne kodowanie znaków
  • Meta viewport jest kluczowy dla responsywności
  • Sekcja BODY zawiera całą widoczną zawartość
  • Semantyczne tagi polepszają strukturę dokumentu

Poznasz różne sposoby formatowania tekstu w HTML: nagłówki, paragrafy, pogrubienia i kursywy.
